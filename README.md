# Godot-FPS-Controller
Simple FPS Controller made in Godot, written in GDscript

Easy to use, MIT License Godot First Person Controller.
Has adjustable values in the script and outside.

<img width="1820" height="999" alt="image" src="https://github.com/user-attachments/assets/04896a80-fe1a-456e-b0ed-ac91cc2ba8b2" />

Use shift to sprint

Use control to crouch

There is a Platform along with a lower platform, under which the player cannot fit, unless they crouch.
